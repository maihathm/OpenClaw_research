# Focus: OpenClaw / Agent Ecosystem Radar

This repo tracks a broader agent-tooling and OpenClaw ecosystem surface.

## What to watch

### 1. New repos
- new OpenClaw-adjacent repositories
- notable new browser-use / agent-browser / agent infra repos
- useful repos for evals, guardrails, automation, or agent ops

### 2. New variants
- forks or variants with meaningful technical differences
- new execution models, transport layers, daemon/session models, sandboxing approaches
- alternative browser-control approaches or repo splits

### 3. New usage patterns
- practical tutorials
- engineering writeups
- workflow improvements
- implementation tricks
- debugging notes
- deployment and automation patterns

### 4. Directly relevant OpenClaw changes
- docs changes
- release notes
- issues / PRs with behavior impact
- ecosystem tooling that changes how OpenClaw should be used

## Filtering rules

Keep only items that add one of:
- a genuinely new repo / release / variant
- a new way to use an existing tool
- a meaningful implementation or architecture change
- a practical technique worth trying
- a bug/failure mode that changes operator behavior

Reject:
- hype posts
- generic AI news without workflow value
- recycled summaries with no new evidence
